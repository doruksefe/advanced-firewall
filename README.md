# Lumiguard v1 - Advanced Firewall

Kendi yazdığım IPv4 ve IPv6 destekli güvenlik duvarı scripti.

## Özellikler
- DDoS ve Brute Force saldırılarına karşı koruma
- Web siteleri / sunucular için tasarlandı
- Yaklaşık 950-1000 satır kod
- 40.000+ karakter

## Uyarı
Bu script **sadece eğitim ve lab ortamları** için geliştirilmiştir.  
Gerçek sistemlerde izinsiz kullanmak yasaktır.
